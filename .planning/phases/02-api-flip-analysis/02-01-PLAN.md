Create `Controllers/BazaarController.cs` with:

1. **GET /api/bazaar** (API-01)
   - Returns current prices for all items
   - Query latest snapshot for each item from database
   - Returns `List<BazaarItemDto>`

2. **GET /api/bazaar/{productId}** (API-02)
   - Returns historical prices for specific item
   - Include latest N snapshots (default 100)
   - Returns `PriceHistoryDto`

3. **GET /api/bazaar/{productId}/history** (API-03)
   - Query params: `start` (DateTime), `end` (DateTime), optional `limit`
   - Returns price history within time range
   - Returns `PriceHistoryDto`

Inject `SkyBazaarDbContext` via constructor. Use EF Core queries with `OrderByDescending` for latest first.

Example query pattern:
```csharp
var latestSnapshots = await _context.Snapshots
    .Include(s => s.BazaarItem)
    .Where(s => s.BazaarItem.ProductId == productId)
    .OrderByDescending(s => s.Timestamp)
    .Take(limit)
    .ToListAsync();
```
